CSS

  • Enter your CSS notes here
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • Inline CSS style sheet is adding directly to the html file and are useful for quick and permanent changes.
  • Internal CSS style sheet is embedding an entire stylesheet directly into the HTML file using a style element in the HEAD element.
  • External CSS style sheet is the most common way, this is when we link a file using LINK.
  • asterix declares that ALL elements will have the style properties.
  • The margin property indicates how much space we want outside the element.
  • The padding property adds space inside the element.
  • A Link element creates the connection to the external file.
  • The rel attribute specifies the relationship between the current document and the linked document resource.
  • The href attribute specifies the location [URL] of the external resource.
  • This means the location is derived from the current location.
  • box-shadow adds shadow effects around the elements frame.
  • Assining colors in CSS can be done in many ways, including by semantic refrences such as GRAY or BLUE.
  • Can also refer to a color by its HEXADECIMAL CODE or hex #.
